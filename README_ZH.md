# AE Crafting Tracker（AE 合成追踪器）

[**English**](README.md)

一个 Minecraft NeoForge 1.21.1 模组，用于实时显示 AE2 网络的合成状态。

## 功能

### Pattern Provider 高亮
- 自动检测 AE2 网络中活跃的 Pattern Provider
- 颜色编码高亮：
  - 🟢 **绿色** — 正在合成
  - 🟡 **黄色** — 卡顿（忙碌但无进展）
  - 🔴 **红色** — 卡死（锁定、输出满或缺材料）
- 在高亮方块上方显示输出物品图标
- 可配置高亮颜色和透明度

### 网络定位器
- 可合成的物品，扫描 AE 网络查找匹配过滤物品的方块
- GUI 中 9 个过滤槽位 — 可从 EMI 拖拽或手动放置
- 对 AE 网络控制器/访问点潜行右键以绑定
- 高亮网络中匹配的方块并显示距离信息
- 跨维度支持（仅在与绑定位置同维度时扫描）
- 即时网络切换检测：清除旧高亮并立即扫描新网络
- 即时丢弃检测：丢弃物品时立即清除所有高亮

### 运行时模式
- 在定位器界面中点击按钮启用运行时高亮
- 关闭定位器 GUI 后高亮依然保持
- 可自由开关，不影响配置设置

## 依赖

- Minecraft 1.21.1
- NeoForge 21.1+
- Applied Energistics 2 (AE2) — 运行时
